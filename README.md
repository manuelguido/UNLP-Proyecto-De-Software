# Proyecto de Software 2019 - Informática UNLP

Es un SPA (Single Page Application) desarrollado en Python con Flask y VueJs para un trabajo de la asignatura "Proyecto de Software" de la Facultad de Informática (UNLP).

El sistema se encarga de manejar los datos de escolaridad de la Orquesta Escuela de Berisso.

Para la base de datos no se uso ningún ORM, dado que no nos lo permitieron, por ende las consultas se realizaron con RAW SQL.

Además, si bien Flask sigue una estructura MVT (Model View Template) se nos pidio seguir el patrón MVC para estructurar el proyecto.
